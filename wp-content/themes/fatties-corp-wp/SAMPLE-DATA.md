## How to Import This Data

### Method 1: Manual Entry (Recommended for Learning)

1. **Add Team Members:**
   - Go to WordPress Admin ‚Üí Team Members ‚Üí Add New
   - For each team member above:
     - Enter name as title
     - Upload photo from `assets/images/`
     - Fill in position and social links
     - Add bio in content editor
     - Click Publish

2. **Add Projects:**
   - Go to WordPress Admin ‚Üí Projects ‚Üí Add New
   - For each project above:
     - Enter project name as title
     - Upload screenshot as featured image
     - Fill in project type and URL
     - Add description in content editor
     - Assign category (Web/App/Graphic)
     - Click Publish

### Method 2: Using WordPress Importer (Advanced)

If you're familiar with WordPress XML import/export, you can create an XML file with this data and import it using the WordPress Importer plugin.
