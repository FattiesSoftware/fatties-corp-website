# Fatties Corporation Theme - Installation Guide

## Quick Start

### Step 1: Install WordPress
Make sure you have WordPress installed and running on your server or local environment.

### Step 2: Upload Theme
1. Compress the `fatties-corp-wp` folder into a ZIP file
2. Go to WordPress Admin ‚Üí Appearance ‚Üí Themes ‚Üí Add New
3. Click "Upload Theme" and select the ZIP file
4. Click "Install Now"
5. After installation, click "Activate"

### Step 3: Set Up Homepage
1. Go to Settings ‚Üí Reading
2. Under "Your homepage displays", select "A static page"
3. Create a new page called "Home" (Pages ‚Üí Add New)
4. Select "Home" as your homepage in Settings ‚Üí Reading
5. Save changes

## Troubleshooting

### Theme not showing correctly?
- Make sure you've set a static homepage (Settings ‚Üí Reading)
- Clear your browser cache
- Check that all required images are uploaded

### Animations not working?
- Make sure JavaScript is enabled in your browser
- Check browser console for errors
- Try disabling other plugins that might conflict

### Statistics not counting up?
- Scroll down to the statistics section
- The animation triggers when you scroll to that section
- Make sure JavaScript is enabled

### Projects/Team members not showing?
- Make sure you've published them (not just saved as draft)
- Check that featured images are set
- Clear WordPress cache if using a caching plugin
